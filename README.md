# MySQL 실습을 위해 만든 간단한 프로젝트입니다.



## 작업 환경은 다음과 같습니다.

- eclipse 2020-12
- jdk 1.8
- tomcat 8.5
- MySQL 5.7.35
- MyBatis 3.4.1
- MyBatis-Spring 1.1.1
- Spring framework 5.1.2.RELEASE
- MySQL workbench 8.0.26



## 구현 기능은 다음과 같습니다.
- MySQL 데이터베이스 정보를 가져와 이름을 입력하면 이름, 나이, 성별, 전부 다 조회하는 버튼으로 확인할 수 있습니다.
